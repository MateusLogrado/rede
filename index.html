Oque é um bit?

O bit é a menor informação que pode ser transmissível e armazenada,
mas claro, como ele é bem pequeno, ele so pode armazenar duas coisas, o numero 0 e o 1,
o 0 seria quando ele esta desligado e 1 quando ele esta ligado.

historia do bit

O termo começou a se popularizar por causa do artigo A mathematical Theory of Communication,
feito pelo engenheiro dos Estados Unidos, Claude Shannon,
ele definiu bit como dois digitos que se juntam para formar uma determinada informação.

oque é um byte?

O byte é uma unidade de armazenamento, se usa nos computadores e celulares modernos,
ele pode ser usado como unidade de medida, para armazenamento ou potencia de processamento

Qual a unidade de medida de transmissão de dados?

Bps ou bit por segundo, é quantos de bit por segundo que podem ser transmitidos
para unidades maiores ele pode ter outros nomes, como:

  • KiloByte = 1024 Bytes
  • MegaByte = 1024 KiloBytes
  • GigaByte = 1024 MegaBytes
  • TeraByte = 1024 GigaBytes
